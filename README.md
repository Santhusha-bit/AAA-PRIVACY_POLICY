# Privacy Policy for Ask Claude Anywhere

**Last updated**: November 2, 2024

## Overview

Ask Claude Anywhere ("the Extension") is committed to protecting your privacy. This policy explains what information we collect (or don't collect) and how it's used.

## What We DON'T Collect

Ask Claude Anywhere does **NOT** collect, store, transmit, or have access to:

- ❌ Personal information
- ❌ Browsing history
- ❌ Conversation history
- ❌ Usage analytics
- ❌ Cookies or tracking data
- ❌ Any user data whatsoever

## What Gets Stored (Locally Only)

The following data is stored **only in your browser** using Chrome's local storage:

1. **Your Claude API Key**
   - Stored locally in your browser
   - Never transmitted to our servers (we don't have servers!)
   - Only sent directly to Anthropic's Claude API when you make requests
   - You can delete it anytime by clearing extension data

2. **Extension Settings**
   - Local preferences only
   - No personal information

**Important**: This data never leaves your device and is never accessible to us or anyone else.

## What Gets Sent to Third Parties

When you use the extension to ask Claude a question, the following is sent:

### To Anthropic Inc. (Claude API)

- The text you selected or typed
- Your API key (for authentication)

This happens when:
- You click "Ask Claude"
- You use the Summarize or Explain buttons
- You right-click and select "Ask Claude about this"

**Details:**
- Communication is direct from your browser to Anthropic
- Encrypted via HTTPS
- Not routed through any of our servers
- Subject to [Anthropic's Privacy Policy](https://www.anthropic.com/privacy)
- Subject to [Anthropic's Terms of Service](https://www.anthropic.com/legal/terms)

**We are not responsible for how Anthropic processes your data.** Please review their privacy policy.

## How Your API Key is Used

Your Claude API key is used solely to:

1. Authenticate your requests to the Claude API
2. Enable the extension to work on your behalf

Your API key:
- ✅ Stays in your browser's local storage
- ✅ Is sent only to Anthropic's API (encrypted via HTTPS)
- ✅ Is never sent to our servers
- ✅ Is never shared with anyone
- ✅ Can be deleted anytime by you

## Security

We implement security best practices:

- ✅ **HTTPS Only**: All API communications are encrypted
- ✅ **No Storage**: We don't store anything on our end
- ✅ **Local Only**: Data stays in your browser
- ✅ **No Tracking**: No analytics or tracking code
- ✅ **Open Source**: Code can be reviewed on GitHub

## Your Rights
